Where would you go if you had a time machine?

This sounds boring but hear me out.
If I did use the machine, I'd only go a few minutes in either direction. The biggest danger in using a time machine is that it occupies physical space. The universe is in constant motion so predicting what space in the universe that machine would end up occupying would be imperative. In all probablity, you would end up in (ahem) space.
Using a heliocentric analogy, with the Earth's rotation your machine would most likely end up in an ocean if you didn't calculate correctly.
